# Libro Portfolio – Esperienza Web Interattiva

Applicazione web **front-end** che riproduce un **libro 3D sfogliabile**, sviluppata con tecnologie **vanilla** (HTML, CSS e JavaScript) e focalizzata su **animazioni avanzate**, **resa visiva** e **controllo diretto del rendering**.

Il progetto è concepito come **case study tecnico da portfolio**, non come semplice demo grafica.

---

## Descrizione

L’applicazione simula un **libro fisico** con pagine sfogliabili, animazioni cinematiche e gestione avanzata del layout su **desktop e mobile**.

La scelta di **non utilizzare framework moderni** è intenzionale e serve a dimostrare la padronanza diretta del **DOM**, delle **animazioni CSS** e della **logica JavaScript**, senza astrazioni.

---

## Funzionalità principali

- **Libro 3D interattivo** con Turn.js  
- **Animazioni CSS complesse** (materializzazione, glow, fumo, ombre)  
- **Rendering fluido** con sincronizzazione JavaScript + CSS  
- **Layout responsive**  
- **Gestione orientamento mobile** (portrait e landscape)  
- **Sistema di accesso con password lato client**  
- **Asset ottimizzati** (PNG, JPG, WebP)  
- **Deploy automatico** tramite GitHub e Vercel  

---

## Obiettivi tecnici

- Dimostrare competenze di **sviluppo frontend senza framework**
- Gestire **animazioni e trasformazioni 3D avanzate**
- Affrontare **edge case mobile** (viewport, resize, orientamento)
- Strutturare un **progetto statico pronto per ambienti cloud**
